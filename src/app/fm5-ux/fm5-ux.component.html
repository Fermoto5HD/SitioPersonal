FM5UX v3

Anteriormente llamado FM5strap

El framework frontend desarrollado a medida para mis sitios y proyectos personales.

CSS CSS minificado

¿Para qué lo hice?

La necesidad de tener un framework propio surgió por tener sitios separados que compartían el mismo layout base: Un Bootstrap v4 con una capa de estilos de muy pocas líneas.

Entonces decidí juntar todo eso dentro de un framework visual y a medida de mis necesidades.
