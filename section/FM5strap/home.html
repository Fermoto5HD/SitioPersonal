FM5strap v2.0.0

Es un framework propio basado en Bootstrap v4 que utilizo para crear páginas propias, como webapps y proyectos.

CSS CSS minificado

¿Porqué un framework (casi) propio?

Dado a la constante reutilización de estilos que tenía entre un proyecto y otro decidí crear, usando como base a Bootstrap v4, un framework que englobe esas reutilizaciones y más.

Componentes

El framework posee algunos componentes adicionales y algunos workarounds para lograr que los componentes originales tengan una terminación más agradable a la vista.
