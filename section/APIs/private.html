APIs privadas

Estas APIs son especiales y no pueden ser utilizadas libremente por terceros.

¿Por qué?

Pasa lo siguiente: Necesito funciones que se puedan acceder fácilmente para lograr un propósito (ej: puntuar un post).

Para esto debo usar un AJAX request, pero este sólo puede ser levantado por el sitio, no por cualquiera ya que permitiría un abuso de la API alterando la información de la base de datos.

Asi que esta sección la armaste para decir que no vas a liberar estos métodos específicos?

Así es!
