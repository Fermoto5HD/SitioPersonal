Portfolios destacados

Club Baldomero: Sitio principal

Abril-Mayo 2015

Este sitio fue propuesto como objetivo para los ingresantes al club de programación por los mentores del club de la Biblioteca Baldomero.

Este fue uno de los primeros sitios en realizarse bajo un layout meramente desde cero.

En un principio se realizó el mismo layout propuesto por los mentores,. Durante el transcurso de su realización lo transformé en responsive y le dí funcionalidad bajo Ajax.

Pero más adelante decidí adaptarlo al nuevo diseño del sitio oficial del Gobierno de la Ciudad de Buenos Aires estrenado en abril de 2015.

Factor RH Creativo

Rebranding Mayo 2015

El cliente solicita la renovación del primer diseño realizado de su sitio web.

Para ello decido basarme en el layout del sitio del club de programación, se adaptaron varios elementos a los colores seleccionados por los clientes.

Dicho sitio se adaptó correctamente a los lineamientos responsive y se optimizaron varias funciones internas del sitio que antes estaban desordenadas.

El sitio continúa bajo desarrollo.

Club Baldomero: Tutorial Formularios

Abril-Mayo 2015

El objetivo de este sitio ha sido el de crear un tutorial sencillo para la creación de formularios html para contacto.

Este sitio ha seguido el mismo layout del sitio principal del Club Baldomero, siendo también un sitio responsive.

El hecho de que sea responsive permite que el usuario pueda aprender a usar forms incluso desde cualquier dispositivo móvil.
