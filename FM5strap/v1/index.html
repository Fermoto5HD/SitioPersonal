FM5strap v1.1.6

Es un framework propio basado en Bootstrap v4 que utilizo para crear páginas propias, como webapps y proyectos.

¿Porqué un framework (casi) propio?

Dado a la constante reutilización de estilos que tenía entre un proyecto y otro decidí crear, usando como base a Bootstrap v4, un framework que englobe esas reutilizaciones y más.

Estilo base

Como fuente tipográfica se utiliza, como fuente primaria "Gidole", y como secundaria "Open Sans". Luego continúan las diversas fuentes de sistema, en caso que hayan problemas en bajar las dos primeras fuentes.

Los colores son oscuros para facilitar la navegación nocturna o en entornos de baja iluminación.

El body tiene un color oscuro como fondo, ya que, en caso de tener un elemento fuera del área de navegación, no generará un fondo blanco al costado. También permite que el footer tenga un efecto "sticky".

<main> tag

Esta etiqueta, aparte de diferenciar el header y el footer, aplica un tono gris oscuro, diferenciandose del fondo el cual es más oscuro.

En caso de que el sitio contenga la clasenavbar-fixed-top en el navbar tendrás que agregar la clase content-w-fixed-navbar al <main> para que se adapte en tamaño.

<footer> tag

El <footer> tiene su estilo. Pensado para albergar una descripción del trabajo, la licencia CC, integrantes, links y datos adicionales.

En esta demo no se notará mucho el fondo, ya que está dentro de <main>.

Jumbotron

Es un contenedor que explica, en pocas palabras, una sección o forma de funcionamiento de algo.

Su forma de uso es similar al clásico Jumbotron de Bootstrap, sólo que acá se agrega un <jumbotron-container> como elemento superior, haciendo que el <jumbotron> actúe como sombra
